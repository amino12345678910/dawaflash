## Production Notes

### Required Footage

1. **Screen Recordings:**
   - [ ] Full onboarding flow (CIN → Face → CNAM)
   - [ ] Claims processing flow (receipt → approval)
   - [ ] WhatsApp interface (clean, no sensitive data)

2. **B-Roll:**
   - [ ] Tunisian pharmacy exterior
   - [ ] People using phones (stock footage OK)
   - [ ] Tunisian landmarks (optional, for branding)

3. **Graphics:**
   - [ ] Logo animations
   - [ ] Architecture diagrams
   - [ ] Performance metrics charts
   - [ ] Fraud detection infographic

4. **Text Overlays:**
   - [ ] Statistics
   - [ ] Feature highlights
   - [ ] Technical specs
   - [ ] Contact information

### Technical Requirements

- **Resolution:** 1080p minimum (4K preferred)
- **Aspect Ratio:** 16:9 (YouTube, presentations)
- **Frame Rate:** 30fps or 60fps
- **Audio:** Clear voiceover + background music
- **Captions:** English subtitles (accessibility)
- **Length:** Target 3-4 minutes (max 5)

### Recording Tips

1. **Phone Screen:**
   - Use screen recording software (OBS, QuickTime)
   - Clean phone UI (no notifications)
   - High brightness for visibility

2. **Voiceover:**
   - Clear, professional microphone
   - Calm, confident tone
   - Moderate pace (not too fast)

3. **Animations:**
   - Use After Effects, Premiere Pro, or similar
   - Smooth transitions
   - Consistent branding (colors, fonts)

4. **Music:**
   - Royalty-free (YouTube Audio Library, Epidemic Sound)
   - Volume: Background level (don't overpower voice)
